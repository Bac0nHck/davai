##Lua скрипт для Roblox, который был сделан по приколу, а точнее - по просьбе подписчика (https://imgur.com/a/megpjho)
```
Или если тебя убивают то тебе говорят давааай ты сможеешь верь в себя
```
был разработан владельцем t.me/arceusxcommunity

**Данный скрипт ничего особенно не делает, кроме как при смерте отображает рандомную аниме-картинку и звук из мема "Давай-давай-давай, ты сможешь". Сам скрипт с открытым кодом, поэтому вы можете добавлять в него новые картинки/аудио, либо заменять прошлые**
- Плюсы данного скрпита в том, что:
- вы всегда будете в тонусе;
- у вас будет личная поддержка;
- о вас есть кому позаботиться;
- а что потом?;
**Так что, если у вас потная каточка, либо просто скучно, то используйте данный скрипт, во!**
